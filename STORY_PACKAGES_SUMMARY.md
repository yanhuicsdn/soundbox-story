## 📊 数据结构

每个故事包JSON文件包含：

```json
{
  "package_name": "包名称",
  "package_id": "包ID (sleep/brave/emotion)",
  "age_group": "适合年龄",
  "total_duration": 总时长(分钟),
  "story_count": 故事数量,
  "target_duration": 目标时长(120分钟),
  "stories": [
    {
      "cid": "故事ID",
      "name": "故事名称",
      "content": "故事完整内容",
      "word_count": 字数,
      "estimated_duration": 预计时长(分钟),
      "tags": ["匹配的关键词标签"]
    }
  ]
}
```

## 🎯 使用方式

### 1. 读取故事包
```javascript
const sleepPackage = require('./story-packages/sleep.json');
console.log(`共${sleepPackage.story_count}个故事，总时长${sleepPackage.total_duration}分钟`);

// 遍历故事
sleepPackage.stories.forEach(story => {
  console.log(story.name);
  console.log(story.content); // 故事内容
  console.log(`预计${story.estimated_duration}分钟`);
});
```

### 2. AI语音生成
使用这些故事内容配合你的Index TTS模型批量生成音频：
- 输入：故事文本内容
- 输出：家长声音的音频文件
- 格式：每个故事生成独立的音频文件

### 3. 网站展示
在网站上展示每个故事包的内容：
- 故事名称列表
- 每个故事的简介
- 预计时长
- 样本音频（可先生成3-5个示例）
