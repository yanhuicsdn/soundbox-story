## 🎨 自定义指南

### 修改品牌颜色
编辑 `styles.css`:
```css
:root {
    --primary-color: #FF6B6B;  /* 改成你喜欢的颜色 */
}
```

### 修改价格
编辑 `script.js`:
```javascript
const products = {
    sleep: { name: '哄睡故事包', price: 79 }  // 修改价格
};
```

### 修改录音文本
编辑 `index.html` 的 `<div class="recording-text">` 部分

### 修改联系方式
编辑 `index.html` 的Footer部分
