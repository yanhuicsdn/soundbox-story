## 🎨 部署后需要做的事情

### 1. 准备示例音频(重要!)

目前网站上的示例音频是占位符,你需要:

- 用你自己的AI声音克隆技术生成1-2个示例故事
- 命名为:
  - `demo-mom.mp3` (妈妈的声音示例)
  - `demo-dad.mp3` (爸爸的声音示例)
- 将这两个文件放到 `~/soundbox-story/` 目录
- 重新运行部署命令更新网站

### 2. 修改联系方式

编辑 `~/soundbox-story/index.html` 第371-373行:

```html
<a href="mailto:YOUR_EMAIL">YOUR_EMAIL@qq.com</a>
<a href="YOUR_WECHAT">你的微信号</a>
```

### 3. 配置支付接口

编辑 `~/soundbox-story/script.js` 第318-326行:

```javascript
if (paymentMethod === 'wechat') {
    // 替换为你的微信支付链接
    window.location.href = 'YOUR_WECHAT_PAY_URL';
} else {
    // 替换为你的支付宝链接
    window.location.href = 'YOUR_ALIPAY_URL';
}
```

### 4. 配置后端API(如果需要)

编辑 `~/soundbox-story/script.js` 第277-295行,配置你的后端接口。

## 🆘 遇到问题?

### 问题1: 网站打不开

- 检查网址是否正确
- 等待1-2分钟(CDN缓存)
- 清除浏览器缓存

### 问题2: 录音功能不工作

- 确认使用HTTPS(不是HTTP)
- 允许浏览器访问麦克风
- 检查浏览器是否支持(推荐Chrome)

### 问题3: 样式错乱

- 清除浏览器缓存
- 检查网络连接
- 尝试其他浏览器

### 问题4: 部署失败

- 确认网络连接正常
- 检查文件夹路径
- 尝试其他部署平台
